第一部分：二级菜单 — 信号与时标
1. 跳转路径
主界面 → 点击【信号与时标】卡片 → 进入本页面
2. 功能依据
对照说明书 3.9 节，本板块仅一个原厂功能：时标输出，无其他子项。
3. 分区详细设计
区域 1：顶部状态栏（320×36px）
左侧文字：信号与时标（18px 白色字体）
居中文字：菜单选择 · 待机（14px 白色字体）
右侧：空白（无电量）
区域 2：中部内容区（320×144px）
仅单个功能选项，居中布局：
列表尺寸：全屏宽，高度50px，居中放置
样式：常态白底 + 左侧空心选中圆点；点击后底色浅蓝、圆点变实心
显示文字：时标输出
区域 3：底部按键区
按键顺序：清零、返回、停止、切换、OK
清零、停止：置灰，不可操作
返回：点击退回主界面
切换：切换界面样式
OK：选中后进入时标输出三级页面
4. 交互逻辑
页面默认选中「时标输出」；
点击 OK，进入对应实操页面；
点击返回，回到主界面。

第二部分：三级页面 — 时标输出（Sgl）
1. 跳转路径
信号与时标二级菜单 → 选中「时标输出」→ 点击 OK → 进入本页面
2. 原厂功能说明（说明书 3.9 节）
内置5 档固定时标周期（无自定义，严格原厂档位）：
0.1ms / 1ms / 10ms / 100ms / 1s
操作逻辑：通过切换键循环选择档位；OK 键启动信号输出，停止键关闭输出；
硬件输出：仪器后盖板时标插座输出≥5V 脉冲信号；
清零键：清空当前档位选择、复位输出状态。
3. 分区详细设计
区域 1：顶部状态栏（320×36px）
左侧：时标输出 Sgl（18px 白字）
居中：待机 / 输出中（状态自动切换）
右侧：空白
区域 2：中部核心显示区（320×144px，白底居中布局）
分三行展示，层级清晰：
第一行（14px 辅助文字）：当前时标周期
第二行（24px 超大核心文字）：1 ms（默认初始档位，切换键循环变更 5 档）
第三行（14px 状态文字）：输出状态：关闭（启动后变为「输出状态：开启」）
区域 3：底部按键区（全部启用）
按键功能（严格匹配原厂操作逻辑）：
清零（红字）：复位档位为初始值、关闭信号输出，恢复待机；
返回（深灰）：退出页面，退回「信号与时标」二级菜单；
停止（深灰）：立即终止时标信号输出；
切换（深灰）：循环切换 5 档固定周期（顺序：0.1ms → 1ms → 10ms → 100ms → 1s）；
OK（橙字）：启动当前档位的时标脉冲输出。
4. 完整交互流程
进入页面：默认档位1 ms，状态「待机 / 输出关闭」，机箱绿灯常亮、黄灯熄灭；
按切换：循环切换 5 个原厂固定周期，屏幕数值同步刷新；
选好档位后按OK：启动输出 → 状态栏变为「输出中」，机箱黄灯常亮；
按停止：关闭输出，状态恢复为关闭，黄灯熄灭；
按清零：档位复位、输出关闭，回到初始状态；
按返回：退出当前页面，回到二级菜单。

6. 补充说明（原厂硬性约束）
无自定义周期、无占空比调节，仅保留说明书 5 个固定档位；
信号输出期间，切换键仍可变更档位，输出实时跟随切换；
输出幅度由硬件电路保证≥5V，界面仅做状态显示。
