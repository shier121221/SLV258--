第一部分：力学实验专项（二级菜单页面）
一、跳转路径
主界面 → 点击【力学实验专项】卡片 → 进入本页面
二、功能依据
对应原厂说明书 3.5 / 3.6 / 3.8 / 3.7 / 3.10 五大力学实验功能，从上至下依次为：振子周期、碰撞、重力加速度、直线加速度、角加速度。
三、分区详细设计
区域 1：顶部状态栏（320×36px）
左侧文字：力学实验专项（18px 白色字体）
居中文字：菜单选择 · 待机（14px 白色字体）
右侧：无电量，空白
区域 2：中部列表区（320×144px）
共 5 条功能选项，全屏宽竖向均匀排布
单条列表高度：26px，条目间距：4px
样式：常态白底 + 空心选中圆点；点击后底色变浅蓝、圆点变为实心
列表顺序（从上至下）：
振子周期测量
碰撞实验
重力加速度
直线加速度
角加速度实验
区域 3：底部按键区
清零、停止：置灰禁用
返回：可用（退回主界面）
切换：可用（界面模式切换）
OK：可用（选中功能进入对应三级页面）
四、交互逻辑
点击列表项切换选中状态；
选中目标功能后，按 OK 进入对应实操页面；
按 返回 回到仪器主界面。

第二部分：5 个三级实操页面（按菜单顺序）
页面 1：振子周期测量（对应说明书 3.5 节）
一、跳转路径
力学二级菜单 → 选中【振子周期测量】→ 点击 OK 进入
二、原厂功能说明
搭配单摆 / 弹簧振子 + 光电门，测量振动周期；支持 1~255 次连续测量，存储前 64 组数据；
切换键可循环切换显示：单次周期 → 振动频率 → 总计时长。
三、分区设计
区域 1：顶部状态栏
左侧：振子周期测量（白字）
居中：待机 / 测量中（状态自动切换）
右侧：空白
区域 2：中部显示区（白底，居中布局）
第一行（14px 浅灰）：振子周期 · 单次（切换显示对应模式）
第二行（24px 超大核心数值）：0.00000 S（周期 / 时长，5 位小数）
第三行（14px 浅灰，居右）：NO. 01（测量序号）
区域 3：底部按键（全部启用）
清零：清空所有数据，序号重置为 01
返回：退回力学二级菜单
停止：终止测量，锁定数据
切换：循环切换「周期 / 频率 / 总时长」显示模式 + 翻阅历史数据
OK：重启测量
四、完整交互
振子挡光片经过光电门，自动采集周期，序号递增；
测量中绿灯闪烁，待机绿灯常亮；
按切换切换显示内容，按停止锁存，按清零重置。

页面 2：碰撞实验（对应说明书 3.8 节）
一、跳转路径
力学二级菜单 → 选中【碰撞实验】→ 点击OK → 弹出挡片宽度选择弹窗 → 确认后进入测量页
二、原厂功能说明
适用于气垫导轨完全弹性碰撞实验，双光电门采集数据；
自动显示 4 组时间、4 组速度，切换键循环翻阅所有参数。
三、前置弹窗（全局复用）
居中圆角弹窗，选择挡片宽度，选中后按OK确认进入主界面。
四、分区设计
区域 1：顶部状态栏
左侧：碰撞实验
居中：待机 / 测量中
区域 2：中部显示区
第一行（14px）：弹性碰撞 参数显示
第二行（24px 核心值）：T11 = 0.00000 S（当前参数，切换轮播 T11/T12/T21/T22/V11/V12/V21/V22）
第三行（14px 居右）：NO. 01
区域 3：底部按键（全部启用）
切换：循环切换 8 组实验参数（时间 + 速度）
其余按键功能与通用规则一致

页面 3：重力加速度（对应说明书 3.7 节）
一、跳转路径
力学二级菜单 → 选中【重力加速度】→ 点击OK进入
二、原厂功能说明
配合自由落体实验仪，支持单光电门 / 双光电门两种模式；
点击同步（集成在 OK 键逻辑）释放钢球，自动采集多组下落时间，切换翻阅全部数据。
三、分区设计
区域 1：顶部状态栏
左侧：重力加速度
居中：待机 / 测量中
区域 2：中部显示区
第一行（14px）：自由落体 时间参数
第二行（24px）：T1 = 0.00000 S（轮播 T1/T2/T3… 多组时间）
第三行（14px 居右）：NO. 01
区域 3：底部按键（全部启用）
OK 键：同步释放钢球，启动测量
切换：翻阅多组下落时间数据

页面 5：角加速度（对应说明书 3.10 节）
一、跳转路径
力学二级菜单 → 选中【角加速度实验】→ 点击OK → 弹出角度选择弹窗
二、原厂功能说明
搭配转动惯量 / 气垫转盘；进入前选择原厂固定角度：180° / 360°；
采集转动挡光时间，计算角加速度，支持多组数据存储与翻阅。
三、前置弹窗
角度选择弹窗（二选一：180°、360°），选中确认后进入测量页。
四、分区设计
区域 1：顶部状态栏
左侧：角加速度实验
居中：待机 / 测量中
区域 2：中部显示区
第一行（14px）：转动角度：XXX°（显示已选角度）
第二行（24px）：0.00000 S（转动计时）
第三行（14px 居右）：NO. 01
区域 3：底部按键（全部启用）

以上页面用到 3 类全局弹窗，样式统一：居中圆角弹窗、白底、6px 圆角、点击选择 + OK 确认
