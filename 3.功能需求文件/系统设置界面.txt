一、改版：系统设置与工具（二级菜单 最终版）
1. 跳转路径
主界面 → 点击【系统设置与工具】卡片 → 进入本页面
2. 页面结构划分
中部320×144px区域分为上下两部分：
上区：功能选择栏（单条列表，左对齐，同其他二级菜单样式）
下区：静态参数展示区（左对齐，放置设备名称、版本、电气 / 物理参数，静态文字，不刷新）
3. 分区详细设计
区域 1：顶部状态栏（320×36px）
底色：#3A5F8B，白色字体
左侧：系统设置与工具（18px）
居中：菜单选择 · 待机（14px）
右侧：空白（无电量）
区域 2：中部内容区（320×144px）
① 上半区：功能选择列表（高度 42px，全屏宽、左对齐）
样式与「基础计时、力学专项」二级菜单列表完全一致：
左侧空心选中圆点，文字左排布，白底常态、点击浅蓝高亮；
显示内容：设备自检；
默认选中该选项。
② 下半区：静态参数区（剩余高度，左对齐，12px 浅灰字 #555555）
逐行排版，内容完全沿用你提供的原厂参数，行间距均匀，全屏左对齐：
J0201-LCM 存贮式数字毫秒计
固件版本：V1.0
电源：A.C. 220V（±10％），50Hz（±5％）
环境温度：-10℃～+40℃
相对湿度：≤85％（40℃）
工作时长：支持连续工作
外形尺寸：约 240mm×280mm
整机重量：约 1.0kg
说明：参数为静态文本，上电绘制一次，全程不刷新，不影响屏幕寿命。
区域 3：底部按键区（菜单标准状态）
清零、停止：置灰不可操作
返回：退回主界面
切换：切换界面样式
OK：选中后进入【设备自检】三级页面
4. 交互逻辑
列表项左对齐，点击切换选中状态（本页仅 1 个选项）；
点击 OK 进入自检三级页；
点击 返回 回到主界面。

二、改版：设备自检（三级页面 纯功能版）
1. 跳转路径
系统设置与工具（二级）→ 选中设备自检 → 点击 OK 进入
2. 页面说明
移除所有参数内容，仅保留原厂光电门 + 整机自检功能，恢复为标准实操页面样式，和其他三级页面风格统一。
3. 分区详细设计
区域 1：顶部状态栏
左侧：设备自检（18px 白字）
居中：检测中 / 检测完成（状态自动切换）
右侧：空白
区域 2：中部显示区（320×144px 白底，居中布局）
第一行（14px）：光电门 & 整机自检
第二行（24px 核心大字）：OK / GM1 ERROR / GM2 ERROR
第三行（14px，故障专属）：请检查传感器连接（正常状态隐藏）
区域 3：底部按键区（全部启用）
清零：重新启动自检
返回：退回上一级（系统设置二级菜单）
停止：终止当前检测
切换：界面样式切换
OK：再次执行自检
4. 交互 & 硬件联动
进入页面自动启动自检，状态栏同步状态；
自检正常：显示OK，机身绿灯常亮；
光电门故障：显示对应错误提示，机身红灯亮起；
清零 / OK 可重复检测，返回键退出页面。
