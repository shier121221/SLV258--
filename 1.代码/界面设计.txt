页面 1：三级页面 - 遮光计时（S1）
一、跳转路径
主界面 → 基础计时类（二级菜单）→ 选中【遮光计时】→ 点击OK → 进入本页面
二、功能依据（原厂说明书 3.3 节）
适配双光电门，挡光片遮挡光电门自动触发计时；支持1~255 次连续实验，自动存储前 64 组数据；停止键锁存数据，切换键翻阅历史记录，清零一键重置。
三、分区域详细设计
区域 1：顶部状态栏（320×36px）
底色：#3A5F8B，文字纯白色
左侧（X=10）：遮光计时 S1（18px）
居中：设备状态（待机 / 测量中）（14px）
右侧：无电量显示，保持空白
区域 2：中部核心显示区（320×144px，白底 #FFFFFF）
整体居中布局，分三行展示，间距均匀：
第一行（14px 浅灰字）：检测光电门 GM1 / GM2（提示触发源）
第二行（24px 超大号深灰字，核心数值）：0.00000 S（计时值，保留 5 位小数，匹配毫秒级精度）
第三行（14px 浅灰字，居右下角）：NO. 01（当前测量序号，范围 1~255）
区域 3：底部按键区（全按键启用）
按键顺序：清零、返回、停止、切换、OK
清零（红字）：清空所有数据，计时归 0、序号重置为 01
返回（深灰）：退出当前页面，回到【基础计时类】二级菜单
停止（深灰）：终止测量，锁定当前计时数据，不再响应光电门触发
切换（深灰）：翻阅已存储的 64 组历史遮光数据（上一组 / 下一组切换）
OK（橙字）：重新启动测量，解除锁定，恢复光电门触发
四、完整交互流程
进入页面：默认显示 0.00000 S、NO.01，状态栏显示「待机」，绿灯常亮；
遮挡任意光电门 → 自动计时，状态栏变为「测量中」，绿灯闪烁；
挡光结束 → 计时自动锁存，序号 + 1，等待下一次测量；
按下停止：暂停测量，光电门失效；
按下切换：循环查看历史 64 组数据；
按下清零：全部重置，回到初始状态；
按下返回：退回二级菜单。

页面 2：三级页面 - 间隔计时（S2）
一、跳转路径
主界面 → 基础计时类（二级菜单）→ 选中【间隔计时】→ 点击OK → 先弹出挡光框选择弹窗 → 确认后进入本测量页
二、功能依据（原厂说明书 3.4 节）
使用挡光框测量两次挡光的时间间隔；进入测量前必须选择挡光框宽度；同样支持多组测量、存储前 64 组数据，操作逻辑与遮光计时一致。
三、前置弹窗：挡光框宽度选择（全局复用弹窗）
弹窗样式：居中圆角弹窗，白底、6px 圆角；
选项：预设多档挡光宽度（适配实验配件）；
操作：点击选中宽度 → 按OK确认 → 进入间隔计时主页面。
四、分区域详细设计
区域 1：顶部状态栏（320×36px）
左侧：间隔计时 S2（18px 白字）
居中：待机 / 测量中（状态自动切换）
右侧：无电量
区域 2：中部核心显示区（320×144px，白底）
布局与遮光计时一致，仅文字微调：
第一行（14px）：挡光间隔计时（功能说明）
第二行（24px 核心数值）：0.00000 S（间隔时间，5 位小数）
第三行（14px 右下角）：NO. 01（测量序号）
区域 3：底部按键区（全按键启用）
清零：清空间隔数据、序号重置
返回：退回二级菜单
停止：终止测量，锁存数据
切换：翻阅 64 组历史间隔数据
OK：重启测量
五、完整交互流程
选中功能按 OK → 弹出挡光框宽度选择，选定后进入测量页；
挡光框两次遮挡光电门 → 自动计算并显示时间间隔；
序号自动累加，支持连续测量；
停止 / 切换 / 清零 / 返回 逻辑与遮光计时完全一致。

页面 3：三级页面 - 计数模式（C）
一、跳转路径
主界面 → 基础计时类（二级菜单）→ 选中【计数模式】→ 点击OK → 进入本页面
二、功能依据（原厂说明书 3.2 节）
纯计数功能，遮挡任意光电门一次，计数值 + 1；无计时功能，仅统计挡光次数；停止锁定计数，清零归零。
三、分区域详细设计
区域 1：顶部状态栏（320×36px）
左侧：计数模式 C（18px 白字）
居中：待机 / 计数中
右侧：无电量
区域 2：中部核心显示区（320×144px，白底）
侧重计数值展示，弱化时间单位，布局：
第一行（14px）：光电门计数（功能提示）
第二行（24px 超大号数字，核心）：0（累计计数，范围 0~99999999）
第三行（14px）：当前次数（辅助标注）
区域 3：底部按键区（全按键启用）
清零：计数值直接归零
返回：退回二级菜单
停止：停止计数，锁定当前数值（不再累加）
切换：本模式无历史数据，点击无额外动作（可保留界面切换功能）
OK：解除锁定，重新开始计数
四、完整交互流程
进入页面，初始计数为 0，状态栏「待机」；
遮挡光电门 → 数字 + 1，状态栏变为「计数中」；
按下停止：计数冻结，不再变化；
按下清零：数值归零；
按下OK：解除冻结，继续计数；
按下返回：退回二级菜单。

页面 4：三级页面 - 通用计时（拓展手动计时）
一、跳转路径
主界面 → 基础计时类（二级菜单）→ 选中【通用计时】→ 点击OK → 进入本页面
二、功能说明
原厂无专用模式，为拓展纯手动计时功能；不依赖光电门，完全通过按键控制启停，适用于无传感器的普通计时场景，计时范围 0.00ms ~ 9999999s。
三、分区域详细设计
区域 1：顶部状态栏（320×36px）
左侧：通用计时（18px 白字）
居中：待机 / 计时中
右侧：无电量
区域 2：中部核心显示区（320×144px，白底）
纯手动计时布局：
第一行（14px）：手动启停计时（提示无光电触发）
第二行（24px 核心数值）：0.00000 S（计时时长）
第三行：无额外编号（纯计时，无多次分组）
区域 3：底部按键区（全按键启用）
清零：计时归 0
返回：退回二级菜单
停止：手动暂停计时
切换：界面模式切换
OK：启动 / 继续计时（核心启停键）
四、完整交互流程
进入页面，初始 0.00000 S，待机状态；
按OK：开始计时，状态栏变为「计时中」；
按停止：计时暂停，数值冻结；
再次按OK：继续计时；
按清零：时长归零；
按返回：退出页面。

补充：统一开发 & 适配要点（四页面通用）
屏幕刷新：全部采用 ST7789V 局部刷新，仅更新变化的数值 / 状态，禁止全屏刷新，杜绝闪屏；
数据精度：计时类页面统一保留5 位小数，匹配仪器 0.00ms 的最小精度；
数据存储：遮光 / 间隔计时自动缓存前 64 组数据，切换键实现上下翻页；计数 / 通用计时无分组存储；
触控：所有点击动作保留 20ms 防抖，适配电阻屏特性；
弹窗复用：间隔计时的「挡光框选择弹窗」为全局弹窗，后续实验功能可直接复用。
